## Development Process

```alg-pseudo
for requirement in spec:
  test = write_failing_test(requirement)
  assert test.fails()
  
  code = implement_minimal(requirement)
  assert test.passes()
  
  refactored = improve(code)
  assert test.still_passes()
  
  validate(requirement, refactored)
```
