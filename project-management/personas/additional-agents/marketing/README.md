# Marketing Agents

Agents focused on go-to-market strategy, customer engagement, and market positioning.

## Agents

### npl-community
Builds and nurtures user communities, manages advocacy programs, and facilitates peer-to-peer learning. Drives word-of-mouth growth and user retention.

### npl-conversion
Optimizes conversion funnels, user activation flows, and monetization pathways. Analyzes drop-off points and designs interventions.

### npl-marketing-copy
Crafts compelling product messaging, feature descriptions, and user-facing communications. Ensures consistency across channels.

### npl-positioning
Develops market positioning strategy, competitive differentiation, and value propositions. Guides strategic messaging across go-to-market initiatives.

## Workflows

**Market Launch Campaign**
1. npl-positioning: Define positioning and key messages
2. npl-marketing-copy: Create marketing materials
3. npl-conversion: Design activation funnels
4. npl-community: Build launch community, manage advocacy

**User Acquisition & Activation**
1. npl-conversion: Identify conversion bottlenecks
2. npl-marketing-copy: Refine messaging for target segments
3. npl-community: Engage early adopters
4. Iterate based on metrics

## Integration Points

- **Upstream**: prd-editor (feature specs), npl-thinker (strategy)
- **Downstream**: npl-user-researcher (feedback loops), npl-community (retention)
- **Cross-functional**: npl-user-impact-assessor (prioritization), npl-onboarding (UX)

## Key Responsibilities

| Agent | Primary | Secondary |
|-------|---------|-----------|
| **community** | Community building, user advocacy | Event management, peer support |
| **conversion** | Funnel optimization, activation | A/B testing, analytics interpretation |
| **marketing-copy** | Messaging, copywriting, consistency | Feature positioning, use case articulation |
| **positioning** | Market positioning, differentiation | Competitive strategy, value prop |
