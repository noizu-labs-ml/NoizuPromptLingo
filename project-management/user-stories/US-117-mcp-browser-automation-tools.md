## Story

As a **QA tester**,
I want **to automate browser interactions for screenshot comparison and form testing via MCP tools**,
So that **visual regression testing can be automated through MCP workflows**.

## Acceptance Criteria

- [ ] Can capture screenshots with viewport and theme control
- [ ] Can compare screenshots with configurable threshold
- [ ] Can automate form submission
- [ ] Can navigate and interact with web pages
- [ ] Can manage browser session state
- [ ] Can inject scripts and styles
