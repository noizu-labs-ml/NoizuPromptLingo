## Overview

NoizuPromptLingo uses a layered architecture where markdown-based definitions flow through interface tools to LLM consumers.

## Layers

### Consumer Layer
Claude Code, LLM clients, and MCP-compatible tools that consume NPL definitions.

### Interface Layer
- **MCP Server**: 23 tools for artifacts, reviews, chat
- **CLI Scripts**: npl-load, dump-files, git-tree
- **CLAUDE.md**: Project configuration

### Definition Layer
- **NPL Syntax**: Core language definitions in `/npl/`
- **Agents**: 45+ agent definitions in `/core/agents/`
- **Personas**: AI identity definitions

### Storage Layer
SQLite databases for artifacts, chat, reviews, and knowledge bases.
