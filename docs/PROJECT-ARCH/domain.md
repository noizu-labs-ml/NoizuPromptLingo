## Overview

The project operates across three bounded contexts: NPL Framework, MCP Tooling, and Personas.

## Bounded Contexts

### NPL Framework
Core prompt syntax and agent definitions.

**Entities**:
- Agents (behavior definitions)
- Directives (instruction patterns)
- Prefixes (response modes)
- Pumps (reasoning patterns)
- Fences (code section types)

### MCP Tooling
Runtime artifact and collaboration management.

**Entities**:
- Artifacts (versioned content)
- Revisions (version history)
- Reviews (inline comments, annotations)
- ChatRooms (collaboration spaces)
- Notifications (@mentions, events)

### Personas
AI identity management for multi-agent workflows.

**Entities**:
- Personas (identity definitions)
- Teams (persona groups)
- Journals (experience tracking)
- Knowledge Bases (persona memory)
