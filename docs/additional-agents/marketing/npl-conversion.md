# npl-conversion

Growth and retention optimization specialist that analyzes user behavior patterns, creates viral mechanics, optimizes conversion funnels, and develops data-driven strategies for sustainable user acquisition and engagement.

## Purpose

Analyzes conversion funnels, designs viral growth mechanics, and builds retention strategies focused on developer audiences. Applies product-led growth methodologies with statistical rigor and developer psychology principles for sustainable conversion optimization.

## Capabilities

- Analyze user behavior patterns and identify conversion bottlenecks
- Design viral acquisition mechanics and referral systems
- Optimize conversion funnels for developer audiences
- Build retention strategies and habit formation frameworks
- Create A/B test plans with statistical significance
- Track conversion metrics across user journey stages
- Identify leading indicators for predictive optimization
- Develop engagement loops and expansion opportunities

## Usage

```bash
# Analyze conversion funnel
@npl-conversion analyze funnel --stage="trial-to-paid" --timeframe="last-90-days"

# Design viral growth mechanics
@npl-conversion create viral-loop --trigger="template-creation" --measurement-plan

# Optimize retention strategy
@npl-conversion design retention --focus="day-7-drop-off" --intervention="onboarding"
```

## Workflow Integration

```bash
# Optimize copy based on conversion data
@npl-conversion identify friction-points > barriers.md && @npl-marketing-copy optimize copy.md --barriers.md

# Community-driven viral mechanics
@npl-conversion design viral-loops --community-engagement && @npl-community implement features

# Test and grade optimization strategies
@npl-conversion plan experiment --hypothesis="simplified-onboarding" && @npl-grader evaluate test-plan.md
```
