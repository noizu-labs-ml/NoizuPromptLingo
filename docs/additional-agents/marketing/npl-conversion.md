# npl-conversion

Growth and retention optimization agent for developer products. Analyzes user behavior, designs viral mechanics, optimizes funnels, and builds data-driven retention strategies.

For complete documentation, see [npl-conversion.detailed.md](./npl-conversion.detailed.md).

## Capabilities

- Analyze conversion funnels and identify bottlenecks
- Design viral acquisition mechanics and referral systems
- Build retention strategies with habit formation frameworks
- Plan A/B tests with statistical significance
- Track metrics across user journey stages

See: [Capabilities](./npl-conversion.detailed.md#capabilities)

## Quick Start

```bash
# Analyze funnel bottlenecks
@npl-conversion analyze funnel --stage="trial-to-paid" --timeframe="90d"

# Design viral growth loop
@npl-conversion create viral-loop --trigger="template-creation"

# Optimize retention
@npl-conversion design retention --focus="day-7-drop-off"

# Plan A/B test
@npl-conversion plan experiment --hypothesis="simplified-onboarding"
```

See: [Usage Examples](./npl-conversion.detailed.md#usage-examples)

## Best Practices

- Measure baselines before optimizing
- Test with statistical rigor; adequate sample sizes
- Focus on retention before acquisition
- Developer audiences require authentic value, not growth hacks

See: [Best Practices](./npl-conversion.detailed.md#best-practices), [Anti-Patterns](./npl-conversion.detailed.md#anti-patterns)
