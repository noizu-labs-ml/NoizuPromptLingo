# Stream Comparison & Selection

Use this prompt to compare the three income streams and make your choice.

---

## PROMPT: Compare Streams for My Situation

I'm trying to decide between AI Templates, Content Publishing, and Print on Demand. Help me compare them for my specific situation.

### My Profile:
- Primary skill/strength: [e.g., "excellent technical writer", "software engineer", "designer"]
- Time available per week: [hours]
- Budget for ads/testing: [amount or "none"]
- Need revenue by: [date or "flexible"]
- Audience size: [e.g., "none", "small email list", "1K Twitter followers"]
- Risk tolerance: [Conservative/Moderate/Aggressive]

### What I Want to Understand:

1. **Effort & Timeline Comparison**
   - Which stream gets me to first revenue fastest?
   - Which requires the least weekly effort long-term?
   - Which requires most upfront effort to get started?

2. **Revenue Potential**
   - What's realistic Year 1 revenue for each, given my situation?
   - How long does each take to reach meaningful income ($1-5K/month)?
   - Which has the highest ceiling long-term?

3. **Audience Requirements**
   - Which needs the biggest audience to be viable?
   - Which can work with a small, niche audience?
   - Do I need to build an audience first for any?

4. **Ad Spend Worthiness**
   - Which streams make sense to invest in paid ads?
   - What CAC (customer acquisition cost) is sustainable?
   - Which ROI the fastest on ad spend?

5. **Sustainability**
   - Which is most passive (truly set-and-forget)?
   - Which requires ongoing maintenance?
   - Which has the longest shelf-life (content stays relevant)?

6. **My Personalized Comparison**

Create a detailed comparison showing:

| Factor | AI Templates | Content Publishing | Print on Demand |
|--------|---|---|---|
| Time to first sale | | | |
| Effort to get started | | | |
| Weekly maintenance | | | |
| Audience needed | | | |
| Realistic Y1 revenue | | | |
| Ad spend ROI | | | |
| Best case scenario | | | |
| Worst case risk | | | |

7. **Specific Recommendation**

Given my profile, recommend:
- Primary stream (with justification)
- Why I should start here vs. others
- When/if to add secondary stream
- Whether I should test multiple simultaneously or go deep on one
