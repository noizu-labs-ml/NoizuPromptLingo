## File Purpose Legend

### Root Level Files
- **INDEX.md**: Passive income generation system overview with platform comparison and tracking
- **Output Format Guides** (7 files): Direct implementation guides for specific platforms and deliverables

### ux-ui-agent/ Core Files
- **CORE.md**: Design philosophy, decision framework, quality defaults - the foundational reference
- **MARKETING.md**: Market validation specialist for testing ideas before building
- **WIREFRAMES.md**: Text-based wireframe formats (ASCII, PlantUML Salt, Wireweave, Mermaid)

### ux-ui-agent/ Subdirectories
- **EVAL/**: Quality rubrics, accessibility audits, performance budgets, automated testing
- **OUTPUTS/**: Platform-specific implementation guides (Next.js, HTML/CSS, p5.js, Textual, SVG, Figma)
- **PATTERNS/**: Reusable component library (layout, components, interactions, responsive, accessibility)
- **PROCESS/**: Design methodologies (brief interpretation, design sprint, validation sprint, handoff)
- **STYLES/**: Five visual style archetypes with selection heuristics
