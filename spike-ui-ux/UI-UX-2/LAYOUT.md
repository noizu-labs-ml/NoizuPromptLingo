# UI-UX-2 Layout

## Root Level Files
- **INDEX.md** - System overview with strategic positioning and revenue projections for passive income system
- **AGENT.md** - Master agent definition (may be overview/parent)
- **ai-templates-AGENT.md** - AI template generation agent prompt with detailed prompts for ideation through launch
- **content-publishing-AGENT.md** - Content publishing agent for article ideation, trend/keyword research, and abstracts
- **pod-AGENT.md** - Print-on-demand product generation agent
- **pod-PROMPT-LIBRARY.md** - Image generation prompt patterns and variations for POD design

## Subdirectory: passive-income-agents/

```
passive-income-agents/
├── INDEX.md - System overview focusing on strategic positioning and platform recommendations
│
├── ai-templates/
│   ├── AGENT.md - AI template generation agent (detailed methodology)
│   ├── PROJECT-TRACKER.md - Template product backlog and metrics tracking
│   └── NICHE-RESEARCH.md - Market analysis framework for template niches
│
├── content-publishing/
│   ├── AGENT.md - Content strategy and publication agent
│   ├── PROJECT-TRACKER.md - Article pipeline tracker with performance metrics
│   └── KEYWORD-RESEARCH.md - SEO and trend research methodology
│
└── print-on-demand/
    ├── AGENT.md - POD product generation agent
    ├── PROJECT-TRACKER.md - Design catalog and sales performance tracking
    └── PROMPT-LIBRARY.md - Image generation prompt templates
```

## Organization Notes
- Root-level AGENT files are duplicates of nested directory versions (consolidate to one location)
- Structured around three income streams with clear subdirectories per stream
- Each stream has: AGENT prompts, PROJECT-TRACKER for management, and research/reference guides
