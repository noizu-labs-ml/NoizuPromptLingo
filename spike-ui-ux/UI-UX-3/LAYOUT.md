## File Organization

**Root Level (Process Documents):**
- Process workflow files appear at root (duplicated from ux-ui-agent/PROCESS/)
- INDEX.md contains unrelated passive income content (appears to be misplaced)

**ux-ui-agent/ (Main Agent System):**
- CORE.md, MARKETING.md, WIREFRAMES.md - Core agent capabilities
- Five subdirectories organize specialized knowledge

**Subdirectories:**
- **EVAL/** - Quality measurement and assessment tools
- **OUTPUTS/** - Implementation guides for different target formats
- **PATTERNS/** - Reusable UI component and layout patterns
- **PROCESS/** - Detailed workflow methodologies
- **STYLES/** - Visual style specifications and selection guidance
